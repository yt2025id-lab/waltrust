PROJECT OVERVIEW

Apa itu WalTrust?

Platform verifikasi kredensial Web3 pertama di ekosistem SUI. Institusi menerbitkan credential, user menyimpannya di Walrus, employer memverifikasi on-chain dalam hitungan detik.

🎯
Positioning sebagai Juri: "WalTrust adalah satu-satunya proyek yang membuktikan bahwa Walrus bukan sekadar 'IPFS yang lebih murah' — tapi infrastructure untuk trust layer baru di internet. Ini adalah killer app yang ekosistem SUI butuhkan."

Problem Statement

Credential verification hari ini: lambat (hari–minggu), mahal (biaya administrasi), mudah dipalsukan, dan terpusat di satu institusi. LinkedIn bisa diisi sembarangan, ijazah bisa dipalsukan, sertifikasi expired tidak ter-update.

Solution

Institusi (univ, lembaga sertifikasi) menerbitkan credential sebagai signed attestation di SUI. Dokumen asli disimpan di Walrus. Employer query langsung ke SUI + Walrus via Tatum RPC. Verification: < 3 detik. Biaya: < $0.01.

User Roles & Flow

  • ✓ Lihat dokumen dari Walrus

/// WalTrust — Decentralized Credential Verification System /// Stores credential metadata on-chain, documents on Walrus module waltrust::credential { use sui::object::{Self, UID, ID}; use sui::tx_context::{Self, TxContext}; use sui::transfer; use sui::event; use std::string::{Self, String}; use sui::clock::{Self, Clock}; use sui::vec_map::{Self, VecMap}; // ============ STRUCTS ============ /// Main credential object — transferable NFT public struct Credential has key, store { id: UID, /// Walrus Blob ID — dokumen asli tersimpan di sini walrus_blob_id: String, /// Walrus Blob ID untuk metadata JSON walrus_metadata_blob_id: String, /// Hash SHA-256 dari dokumen (verification layer kedua) document_hash: vector<u8>, /// Tipe credential: "degree" | "certificate" | "work_experience" credential_type: String, /// Nama credential (e.g., "S1 Teknik Informatika") title: String, /// Institusi penerbit (address) issuer: address, /// Nama institusi (display) issuer_name: String, /// Penerima credential recipient: address, /// Nama penerima recipient_name: String, /// Tanggal issued (Unix timestamp ms) issued_at: u64, /// Tanggal expired (0 = tidak expired) expires_at: u64, /// Status: true = valid, false = revoked is_valid: bool, /// Extra fields untuk extensibility extra_fields: VecMap<String, String>, } /// Capability untuk issuer yang terverifikasi public struct IssuerCap has key, store { id: UID, issuer_address: address, issuer_name: String, is_active: bool, } /// Admin capability untuk mengelola issuer registry public struct AdminCap has key, store { id: UID, } // ============ EVENTS ============ public struct CredentialIssued has copy, drop { credential_id: ID, issuer: address, recipient: address, walrus_blob_id: String, credential_type: String, issued_at: u64, } public struct CredentialRevoked has copy, drop { credential_id: ID, revoked_by: address, revoked_at: u64, } public struct CredentialVerified has copy, drop { credential_id: ID, verifier: address, is_valid: bool, verified_at: u64, } // ============ INIT ============ fun init(ctx: &mut TxContext) { // Transfer AdminCap ke deployer let admin_cap = AdminCap { id: object::new(ctx), }; transfer::transfer(admin_cap, tx_context::sender(ctx)); } // ============ PUBLIC ENTRY FUNCTIONS ============ /// Issue credential baru — hanya IssuerCap holder yang bisa public entry fun issue_credential( issuer_cap: &IssuerCap, walrus_blob_id: vector<u8>, walrus_metadata_blob_id: vector<u8>, document_hash: vector<u8>, credential_type: vector<u8>, title: vector<u8>, recipient: address, recipient_name: vector<u8>, expires_at: u64, clock: &Clock, ctx: &mut TxContext ) { assert!(issuer_cap.is_active, 1001); // E_ISSUER_NOT_ACTIVE let issued_at = clock::timestamp_ms(clock); let credential_id = object::new(ctx); let cred_obj_id = object::uid_to_inner(&credential_id); let credential = Credential { id: credential_id, walrus_blob_id: string::utf8(walrus_blob_id), walrus_metadata_blob_id: string::utf8(walrus_metadata_blob_id), document_hash, credential_type: string::utf8(credential_type), title: string::utf8(title), issuer: issuer_cap.issuer_address, issuer_name: issuer_cap.issuer_name, recipient, recipient_name: string::utf8(recipient_name), issued_at, expires_at, is_valid: true, extra_fields: vec_map::empty(), }; // Emit event — Tatum Data API bisa subscribe ke ini event::emit(CredentialIssued { credential_id: cred_obj_id, issuer: issuer_cap.issuer_address, recipient, walrus_blob_id: credential.walrus_blob_id, credential_type: credential.credential_type, issued_at, }); // Transfer credential ke recipient transfer::transfer(credential, recipient); } /// Revoke credential — hanya issuer asli yang bisa public entry fun revoke_credential( issuer_cap: &IssuerCap, credential: &mut Credential, clock: &Clock, ctx: &mut TxContext ) { assert!(issuer_cap.issuer_address == credential.issuer, 1002); assert!(credential.is_valid, 1003); // Already revoked credential.is_valid = false; event::emit(CredentialRevoked { credential_id: object::uid_to_inner(&credential.id), revoked_by: tx_context::sender(ctx), revoked_at: clock::timestamp_ms(clock), }); } /// Public verify — siapa saja bisa query status credential public fun verify_credential( credential: &Credential, clock: &Clock, ctx: &mut TxContext ): bool { let now = clock::timestamp_ms(clock); let not_expired = credential.expires_at == 0 || now < credential.expires_at; let is_valid = credential.is_valid && not_expired; event::emit(CredentialVerified { credential_id: object::uid_to_inner(&credential.id), verifier: tx_context::sender(ctx), is_valid, verified_at: now, }); is_valid } // ============ VIEW FUNCTIONS ============ public fun get_walrus_blob_id(c: &Credential): &String { &c.walrus_blob_id } public fun get_issuer(c: &Credential): address { c.issuer } public fun get_recipient(c: &Credential): address { c.recipient } public fun is_valid(c: &Credential): bool { c.is_valid } public fun get_document_hash(c: &Credential): &vector<u8> { &c.document_hash } }

// app/lib/walrus.ts const WALRUS_PUBLISHER = "https://publisher.walrus-mainnet.walrus.space"; const WALRUS_AGGREGATOR = "https://aggregator.walrus-mainnet.walrus.space"; const STORAGE_EPOCHS = 365; // ~1 tahun export interface WalrusUploadResult { blobId: string; suiObjectId: string; endEpoch: number; isNewlyCreated: boolean; } export interface CredentialMetadata { title: string; credentialType: string; issuerName: string; recipientName: string; issuedAt: string; // ISO date expiresAt: string | null; additionalInfo: Record<string, string>; } /** * Upload file blob ke Walrus * Returns blob ID yang akan disimpan on-chain di SUI */ export async function uploadToWalrus( fileData: Uint8Array | Blob, contentType: string = "application/octet-stream" ): Promise<WalrusUploadResult> { const response = await fetch( `${WALRUS_PUBLISHER}/v1/blobs?epochs=${STORAGE_EPOCHS}`, { method: "PUT", headers: { "Content-Type": contentType }, body: fileData, } ); if (!response.ok) { throw new Error(`Walrus upload failed: ${response.status}`); } const result = await response.json(); // Handle both newlyCreated and alreadyCertified responses if (result.newlyCreated) { return { blobId: result.newlyCreated.blobObject.blobId, suiObjectId: result.newlyCreated.blobObject.id, endEpoch: result.newlyCreated.blobObject.storage.endEpoch, isNewlyCreated: true, }; } else if (result.alreadyCertified) { return { blobId: result.alreadyCertified.blobId, suiObjectId: "", endEpoch: result.alreadyCertified.endEpoch, isNewlyCreated: false, }; } throw new Error("Unexpected Walrus response format"); } /** * Upload metadata JSON ke Walrus * Metadata berisi info credential yang bisa dibaca tanpa buka PDF */ export async function uploadMetadataToWalrus( metadata: CredentialMetadata ): Promise<WalrusUploadResult> { const jsonBytes = new TextEncoder().encode(JSON.stringify(metadata)); return uploadToWalrus(jsonBytes, "application/json"); } /** * Fetch blob dari Walrus by Blob ID * Digunakan di verifier page untuk tampilkan dokumen */ export async function fetchFromWalrus(blobId: string): Promise<Blob> { const response = await fetch( `${WALRUS_AGGREGATOR}/v1/blobs/${blobId}` ); if (!response.ok) { throw new Error(`Walrus fetch failed for blob: ${blobId}`); } return response.blob(); } /** * Fetch metadata JSON dari Walrus */ export async function fetchMetadataFromWalrus( metadataBlobId: string ): Promise<CredentialMetadata> { const blob = await fetchFromWalrus(metadataBlobId); const text = await blob.text(); return JSON.parse(text) as CredentialMetadata; } /** * Compute SHA-256 hash dari file (untuk on-chain verification) */ export async function hashFile(fileData: ArrayBuffer): Promise<Uint8Array> { const hashBuffer = await crypto.subtle.digest("SHA-256", fileData); return new Uint8Array(hashBuffer); }

// app/lib/contract.ts import { Transaction } from "@mysten/sui/transactions"; import { suiClient, CONTRACT } from "./sui-client"; import type { SignAndExecuteTransaction } from "@mysten/dapp-kit"; /** * Issue credential on-chain * Dipanggil setelah Walrus upload selesai */ export async function issueCredentialTx( params: { issuerCapId: string; walrusBlobId: string; walrusMetadataBlobId: string; documentHash: Uint8Array; credentialType: string; title: string; recipient: string; recipientName: string; expiresAt: number; // 0 = no expiry }, signAndExecute: SignAndExecuteTransaction ) { const tx = new Transaction(); tx.moveCall({ target: `${CONTRACT.packageId}::${CONTRACT.credentialModule}::issue_credential`, arguments: [ tx.object(params.issuerCapId), tx.pure.vector("u8", new TextEncoder().encode(params.walrusBlobId)), tx.pure.vector("u8", new TextEncoder().encode(params.walrusMetadataBlobId)), tx.pure.vector("u8", params.documentHash), tx.pure.vector("u8", new TextEncoder().encode(params.credentialType)), tx.pure.vector("u8", new TextEncoder().encode(params.title)), tx.pure.address(params.recipient), tx.pure.vector("u8", new TextEncoder().encode(params.recipientName)), tx.pure.u64(params.expiresAt), tx.object("0x6"), // Sui Clock object (always 0x6) ], }); return signAndExecute({ transaction: tx }); } /** * Query credential object dari SUI via Tatum RPC */ export async function getCredential(credentialObjectId: string) { return suiClient.getObject({ id: credentialObjectId, options: { showContent: true, showOwner: true, showType: true, }, }); } /** * Get semua credential milik wallet address tertentu */ export async function getWalletCredentials(walletAddress: string) { const objects = await suiClient.getOwnedObjects({ owner: walletAddress, filter: { StructType: `${CONTRACT.packageId}::credential::Credential`, }, options: { showContent: true, showType: true }, }); return objects.data; }

app/app/api/upload/route.ts Upload dokumen ke Walrus — server-side untuk keamanan
// app/app/api/upload/route.ts import { NextRequest, NextResponse } from "next/server"; import { uploadToWalrus, uploadMetadataToWalrus, hashFile } from "@/lib/walrus"; export async function POST(req: NextRequest) { try { const formData = await req.formData(); const file = formData.get("document") as File; const metadataStr = formData.get("metadata") as string; if (!file) { return NextResponse.json({ error: "No file" }, { status: 400 }); } // Validasi file type const allowedTypes = ["application/pdf", "image/jpeg", "image/png"]; if (!allowedTypes.includes(file.type)) { return NextResponse.json({ error: "Invalid file type" }, { status: 400 }); } // Max 10MB if (file.size > 10 * 1024 * 1024) { return NextResponse.json({ error: "File too large (max 10MB)" }, { status: 400 }); } const fileBuffer = await file.arrayBuffer(); const fileBytes = new Uint8Array(fileBuffer); // Upload dokumen asli + metadata ke Walrus secara paralel const [docResult, metaResult, hash] = await Promise.all([ uploadToWalrus(fileBytes, file.type), uploadMetadataToWalrus(JSON.parse(metadataStr)), hashFile(fileBuffer), ]); return NextResponse.json({ documentBlobId: docResult.blobId, metadataBlobId: metaResult.blobId, documentHash: Array.from(hash), docEndEpoch: docResult.endEpoch, }); } catch (err) { console.error("Upload error:", err); return NextResponse.json({ error: "Upload failed" }, { status: 500 }); } }

README Template Structure

# WalTrust — Decentralized Credential Verification ## 🏆 Tatum × Walrus Hackathon Submission ### What is WalTrust? WalTrust is a trustless credential verification system built on SUI. Documents live on Walrus. Attestations live on-chain. Verification takes 3 seconds. ### Architecture [Diagram goes here — use draw.io atau Mermaid] SUI Move Contract → stores: credential metadata, blob IDs, validity status Walrus Storage → stores: original PDF/image document + metadata JSON Tatum RPC → powers: ALL on-chain queries and transaction execution Next.js Frontend → provides: issue, verify, and wallet management UI ### Tech Stack - Smart Contract: Sui Move (SUI Mainnet) - Storage: Walrus (Mainnet) ← CORE, not add-on - RPC: Tatum (sui-mainnet.gateway.tatum.io) ← ALL calls - Frontend: Next.js 14 + TypeScript + @mysten/dapp-kit - Hosting: Vercel ### Live Demo 🌐 https://waltrust.vercel.app 📦 Package ID: 0x[mainnet_package_id] 🔍 SuiScan: https://suiscan.xyz/mainnet/object/[package_id] ### How to Run 1. git clone https://github.com/[you]/waltrust 2. cd app && cp .env.example .env.local 3. Fill in TATUM_API_KEY from dashboard.tatum.io 4. npm install && npm run dev ### Walrus Integration Details Every credential stores TWO Walrus blobs: - walrus_blob_id: The actual document (PDF/image) - walrus_metadata_blob_id: JSON metadata for quick display Both Blob IDs are stored on-chain in the SUI Move object. Revocation: 1 on-chain tx. No Walrus update needed — blob stays, chain says REVOKED.
